# Dream Thief — Concept Art Visuals

A visual gallery for the weapons and tools from **Dream Thief**, featuring detailed concept-art style SVGs and simplified clipart icons.

## Contents

- [Gallery](#gallery) — Browse all gear with two view modes
- [Weapons](#weapons) — 5 weapons, each with a unique playstyle
- [Tools](#tools) — 8 tools for utility, combat, and movement

## Gallery

Open `index.html` in any browser to browse the full collection.

Two tabs are available:

| Tab | Description |
|---|---|
| **Visual Designs** | Full-size detailed SVGs with gradients, glows, particles, and environment context |
| **Clipart Icons** | Compact 120×120 inventory-style icons with simplified shapes and solid colors |

Click any card or icon to open a larger view with a blurred-background modal.
